ğŸš€ Quick Start

Prerequisites

  • Node.js 18+ and npm
  • Python 3.11+
  • Cloudflare account with Workers and Pages access
  • GitHub repository with Actions enabled

Setup

# Clone and install dependencies
git clone <repository-url>
cd hackerhardware-net
npm install
pip install -r requirements.txt

# Configure Cloudflare
npx wrangler login
cp wrangler.example.toml wrangler.toml
# Edit wrangler.toml with your account details

# Start development servers
npm run dev:frontend  # Frontend development
npm run dev:backend   # FastAPI backend
npm run dev:worker    # Cloudflare Worker

Deployment

# Deploy to production
npm run deploy:all

# Deploy individual components
npm run deploy:frontend  # Cloudflare Pages
npm run deploy:backend   # Backend API
npm run deploy:worker    # Edge Workers
